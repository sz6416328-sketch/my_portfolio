# Interactive Personal Portfolio 🚀

Welcome to my personal portfolio website! This project is a modern, clean, and fully responsive digital space built to showcase my web development skills, software engineering milestones, and academic background.

## 💻 Tech Stack Used
* **Frontend:** HTML5, CSS3, JavaScript (ES6)
* **Fonts & Icons:** Google Fonts (Poppins), FontAwesome Icons
